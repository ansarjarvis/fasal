# FASAL Movie Open Source

## Prerequisites

- React
- Next.js
- Mongodb
- Prisma
- Clerk ( for auth )
- shadcn ui

## Installing

**1. Clone the repository**

- run `git clone https://github.com/ansarjarvis/fasal.git`

**2. Install dependencies**

- run `npm install`

**3. Run the server**

- run `npm run dev`

## Environment Variables

DATABASE_URL

NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY

CLERK_SECRET_KEY
